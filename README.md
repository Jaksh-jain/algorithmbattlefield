# ⚔️ Algorithm Battlefield

A real-time MCQ Quiz Battle Platform designed for interactive learning, competitive coding practice, and classroom engagement.

---

## 🚀 Overview

Algorithm Battlefield enables users to participate in live quiz battles with real-time synchronization, leaderboard tracking, and engaging UI.

Built for:
- Students practicing DSA & CS concepts
- Teachers conducting live quizzes
- Competitive learning environments

---

## 🎯 Features

### 🧠 MCQ Question Bank
- 160+ curated MCQs
- Topics:
  - Operating Systems
  - Virtual Machines
  - Linux
  - Kubernetes
  - Python
  - Java
  - JavaScript
  - Data Structures & Algorithms
- Question types:
  - Concept-based
  - Output prediction
  - Time complexity
  - Scenario-based

---

### 🏆 Quiz Battle System
- Real-time multiplayer quiz rooms
- 2-minute timer per question
- Animated answer selection
- Instant answer reveal
- Heatmaps showing answer distribution
- Fastest responder highlights

---

### 🔐 Room System
- Topic-based room creation
- Password-protected rooms
- Live participant tracking
- Join/leave in real-time

---

### 🎮 Teacher Controls
- Skip questions
- End quiz anytime
- Monitor live responses
- Control room flow

---

  - 💎 Diamond
